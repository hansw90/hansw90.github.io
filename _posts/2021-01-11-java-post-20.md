### 1. 함수형 인터페이스와 람다 표현식

##### 1-1 함수형 인터페이스 이해하기

함수형 인터페이스 (Functional interface) 는 1개의 추상 메소드를 가지고 있는 인터페이스를 말한다. (= Single Abstract Method(SAM))

함수형 인터페이스 예
```java
public interface FunctionalInterface {
    public abstract void doSomthing(String doIt);
}
```
